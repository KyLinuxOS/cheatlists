## Utiliser cmder

* Faire une variable d'environnement pour le dossier de cmder, par exemple `CMDER_ROOT` qui pointe sur `C:\projects\cmder`
* Mettre `"cmd.exe" /k ""%CMDER_ROOT%\vendor\init.bat""` dans `Settings > Tools > Terminal : Shell path`
