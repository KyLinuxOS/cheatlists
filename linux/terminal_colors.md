# Couleurs dans le terminal

```bash
# Reset
NoColor='\033[0m'       # Text Reset

# Regular Colors
Black='\033[0;30m'        # Black
Red='\033[0;31m'          # Red

Cyan='\033[0;36m'         # Cyan

# Bold
BoldBlack='\033[1;30m'       # Black

# Background

BackgroundWhite='\033[47m'       # White

# High Intensity
HighBlack='\033[0;90m'       # Black
```
